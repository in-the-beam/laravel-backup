# Laravel Backup

Tested on Laravel 5.8

## Requirements

* Archiver
  * tar (with gzip support) - Laravel backup uses command `tar cfvz`
  * zip (optional) - Laravel backup uses command `zip`
  * rar (optional) - Laravel backup uses command `rar`
* Database
  * MySQL 5.5+
  * PostgreSQL 9+
